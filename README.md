Динамика лексических характеристик в романе Л.Н. Толстого «Анна Каренина»

Проект представляет собой количественный анализ лексики романа «Анна Каренина». Текст романа разделен на восемь частей в соответствии с оригинальной структурой произведения. В ходе работы выполнена предварительная обработка текста (очистка, удаление стоп-слов, лемматизация), построена описательная статистика, проведен частотный анализ значимых для сюжета слов — имен героев, топонимов, лексики чувств и религиозной лексики. На основе относительных частот топ-100 лемм построена матрица евклидовых расстояний и проведена иерархическая кластеризация методом Уорда. Анализ подтвердил гипотезу о лексической неоднородности частей романа и позволил выявить две лексические группы, соответствующие «деревенской» и «городской» сюжетным линиям.
